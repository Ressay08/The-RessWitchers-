## 🛠️ Tech Stack

* **Frontend:** HTML5, Tailwind CSS (CDN) for a 3D glassmorphism aesthetic.
* **Backend:** Firebase Realtime Database for the "Live Feedbacks" system.
* **Fonts:** Plus Jakarta Sans for a modern, clean typography.
* **Deployment:** GitHub Pages.

---

## 💬 Live Feedback System

The "Guestbook" section uses **Firebase Compat SDK** to allow users to leave real-time messages. It includes:
* **Real-time updates:** Comments appear instantly without refreshing.
* **Security Rules:** Message length restriction (< 200 chars) and path protection.
* **Glassmorphism UI:** Interactive cards with backdrop-blur effects.
